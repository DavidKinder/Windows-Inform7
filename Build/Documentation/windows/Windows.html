Windows Defender

If you are running Windows Defender then you will find that Inform can be very slow to compile and to search documentation. The problem is that Windows Defender insists on scanning every HTML file that applications open or create. The only solution to this is to exclude from scanning the Inform installation directory, the "My Documents\Inform" directory, and any directories containing Inform projects. This can be done from the "Windows Security" app: go to "Virus and Threat Protection", then "Manage Settings", then "Add or Remove Exclusions".

Dictionaries

To install a new dictionary language for spell-checking, download the appropriate dictionary from LibreOffice. Copy the ".aff" and ".dic" files into the "Dictionaries" sub-directory of your Inform installation. Re-start Inform and open the spell-checker: your new dictionary should be included in the drop-down list of languages.
