Windows Inform 7

This Windows version of Inform 7 was created by David Kinder, based on a design by Graham Nelson, Andrew Hunter and Toby Nelson.

Windows Defender

If you are running Windows Defender then you will find that Inform 7 can be very slow to compile and to search documentation. The problem is that Windows Defender insists on scanning every HTML file that applications open or create. The only solution to this is to exclude from scanning the Inform 7 installation directory, the "My Documents\Inform" directory, and any directories containing Inform 7 projects. This can be done from the "Windows Security" app: go to "Virus and Threat Protection", then "Manage Settings", then "Add or Remove Exclusions".

Dictionaries

To install a new dictionary language for spell-checking, download the appropriate dictionary from LibreOffice. Copy the ".aff" and ".dic" files into the "Dictionaries" sub-directory of your Inform 7 installation. Re-start Inform 7 and open the spell-checker: your new dictionary should be included in the drop-down list of languages.

Home Directory

The directory that Inform uses to store extensions, documentation, etc. (which defaults to the user's "My Documents" folder) can be changed by creating a text file "home.txt" in the same directory as the Inform executable, containing the path to the required home directory.

Inform 6

The bundled Inform 6 compiler is generally the latest release, though sometimes it proves necessary to use a version of Inform 6 with further patches beyond the latest release. There is an active GitHub Inform 6 project which contains the revisions of the Inform 6 compiler used.
