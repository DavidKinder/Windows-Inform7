Windows Inform 7

This Windows version of Inform 7 was created by David Kinder, based on a design by Graham Nelson and Andrew Hunter.

Windows Defender

If you are running Windows Defender (or Microsoft Security Essentials, as it was previously known) then you will find that Inform 7 can be very slow to compile and to search documentation. The problem is that Windows Defender insists on scanning every HTML file that applications open or create. The only solution to this is, if possible, to configure Windows Defender to ignore either all the executables in the Inform 7 installation directory, or to ignore all the directories containing Inform projects, as well as the "Inform" directory under "My Documents", and the Inform 7 installation directory.

Shortcuts

The following keyboard shortcuts are available:
F1 Show the documentation index: this is the same as the "Inform Help" item on the "Help" menu.
F2 Switch the keyboard focus to the search source field on the toolbar.
F3 Switch the keyboard focus to the search documentation field on the toolbar.
F4 Show the whole of the source text in the Source tab. This is equivalent to bringing up the Contents view and clicking on the very top heading.
F5 Build the game and run it: this is the same as the "Go" toolbar command.
F6 Switch the keyboard focus to the opposite pane.
F7 Build the game but do not run it: this is the same as the "Refresh Index" item on the "Play" menu.
F9 Build the game and replay to the current position in the skein: this is the same as the "Replay" toolbar command.

When viewing the contents pane, shift-clicking on a section heading will show the entire source pane with the clicked section heading in view, rather than showing only that section.

Dictionaries

To install a new dictionary language for spell-checking, download the appropriate dictionary from LibreOffice. Copy the ".aff" and ".dic" files into the "Dictionaries" sub-directory of your Inform 7 installation. Re-start Inform 7 and open the spell-checker: your new dictionary should be included in the drop-down list of languages.

Home Directory

The directory that Inform uses to store extensions, documentation, etc. (which defaults to the user's "My Documents" folder) can be changed by creating a text file "home.txt" in the same directory as the Inform executable, containing the path to the required home directory.

Inform 6

The bundled Inform 6 compiler is generally the latest release, though sometimes it proves necessary to use a version of Inform 6 with further patches beyond the latest release. There is an active GitHub Inform 6 project which contains the revisions of the Inform 6 compiler used.
